# Отправщик писем
## Задача
Необходимо отправить письмо на конкретный email-адрес.
## Использование
Для отправки сообщения запустите в консоли команду `python main.py -n Ivan -to mail@mail.ru`, где `Ivan` - имя друга, а `mail@mail.ru` - его почта.
